The DENARIO project:

Abstract

We present DENARIO, an AI multi-agent framework designed to perform end-to-end scientific research. DENARIO can generate ideas, develop research plans, write and execute code, make plots, and write a scientific paper. DENARIO is built as a modular system, and therefore, can perform either very specific tasks, such as generating an idea, or carrying out end-to-end scientific research. In this work, we describe DENARIO and its modules in detail and illustrate its capabilities by showing several AI-generated papers generated by it. These papers cover many scientific disciplines, such as biology, chemistry, medicine, machine learning, and astronomy. We also show how DENARIO can generate paper by combining ideas from different disciplines by showing an AI-paper it wrote, combining methods from quantum physics with cosmological data and machine learning techniques. We also publicly release the code associated with DENARIO at Github. DENARIO can also be run directly on the web.
